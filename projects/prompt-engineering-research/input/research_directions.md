# 研究方向：Prompt 工程对代码生成质量的影响

## 背景

大型语言模型（LLM）在代码生成任务上表现出色，但不同的 Prompt 策略可能显著影响生成质量。

## 研究问题

1. Chain-of-Thought (CoT) 提示是否比 Zero-shot 提示在代码生成任务上表现更好？
2. Few-shot 提示在哪些类型的代码任务上优势最明显？
3. 不同 Prompt 策略的计算成本是否值得？

## 初步假设

CoT 提示在复杂算法任务上优于 Zero-shot，但在简单任务上可能过度设计。

## 预期贡献

1. 对比三种 Prompt 策略的性能差异
2. 分析不同策略的适用场景
3. 提供实用的 Prompt 选择指南
